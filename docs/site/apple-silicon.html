🍎 Apple Silicon

TorrentOS on Apple Silicon

Run TorrentOS on any Apple Silicon Mac — M1 through M5 — powered by the open-source Asahi Linux project.

How does this work?

Apple Silicon Macs use ARM64 chips that are completely different from the x86_64 PCs TorrentOS's ISO targets. The Asahi Linux project reverse-engineered Apple's bootchain and GPU drivers, making Linux actually usable on these machines. TorrentOS builds on top of that work — you first install Arch Linux (Asahi), then our bootstrap script transforms it into a full TorrentOS environment.

ℹ️ You can dual-boot TorrentOS alongside macOS — no need to wipe your Mac. The Asahi installer resizes your macOS partition and adds Linux in the free space.

Supported hardware

All Apple Silicon Macs work — newer chips may have partial GPU acceleration until Asahi drivers mature.

Installation guide

Takes about 20–30 minutes on a fast connection. You'll need at least 25 GB of free disk space.

1

Run the Asahi Linux installer

Open Terminal on your Mac and run this command. It will guide you through resizing your macOS partition and installing Linux in the free space.

curl https://alx.sh | sh

When asked which system to install, choose "Arch Linux (Asahi) — Minimal". This is the base we build on.

2

Boot into Arch Linux

After the installer finishes it will ask you to shut down. Hold the power button on next boot to open the startup options, then select your new Linux installation.

You'll land at a minimal command line. Log in with the credentials the installer set up (usually alarm / alarm).

3

Run the TorrentOS bootstrap script

This one command installs Hyprland, all TorrentOS packages, and the full desktop environment:

bash <(curl -fsSL https://torrentos.org/install-asahi.sh)

The script will ask for confirmation before making any changes. It takes roughly 10–20 minutes depending on download speed.

4

Reboot and enjoy

When the script finishes, reboot your Mac. The greetd graphical login screen will appear and you'll be in TorrentOS.

Press Super (⌘) + T to open a terminal, Super + Space for the app launcher.

Differences from the x86_64 ISO

TorrentOS on Apple Silicon is the same experience — here's what's slightly different:

🌐

Chromium instead of Google Chrome

Google Chrome is only available for x86_64. Chromium — the identical open-source build — is installed automatically. Extensions, profiles, and sync all work the same way.

⌨️

Apple keyboard layout

The Command (⌘) key maps to Super in Linux. All TorrentOS shortcuts use Super, so muscle memory transfers naturally. The Option key maps to Alt.

👆

Trackpad gestures

Multi-touch gestures are configured via libinput-gestures. Three-finger swipe switches workspaces, pinch-to-zoom works in apps. The config lives at ~/.config/libinput-gestures.conf.

🔋

Excellent battery life

Apple Silicon's efficiency cores mean you'll get 10–18 hours of battery on a MacBook Air under typical use — often better than macOS for non-GPU workloads.

Troubleshooting

?

No sound after boot

Run systemctl --user restart wireplumber. If that doesn't help, ensure you have the Asahi audio config: paru -S asahi-audio.

?

Screen tearing / no GPU acceleration

Ensure you have the Asahi mesa driver: sudo pacman -S mesa-asahi-edge. Then log out and back in.

?

Hyprland won't start

Check the log: cat ~/.local/share/hyprland/hyprland.log. Most commonly caused by a missing XDG_RUNTIME_DIR — ensure greetd is running as a system service.

?

Run TorrentOS diagnostics

The built-in doctor command checks all services and hardware:

torrentos-doctor
💬 Still stuck? Open an issue on GitHub and include the output of torrentos-doctor and your Mac model.
