What's inside
Everything you need, nothing you don't.

TorrentOS ships ready to go — no post-install tweaking, no hunting for drivers.

🖥️

Desktop that just works

Hyprland compositor with blur, rounded corners, smooth animations, and a macOS-style dock. Looks great on day one.

🌊

Built for torrents

qBittorrent pre-installed and ready. It's in the name — TorrentOS takes downloading seriously.

🌐

Both browsers included

Firefox and Google Chrome ship out of the box. Use whichever you prefer, or both.

♿

Accessibility first

Screen reader, screen magnifier, high-contrast support, and text-to-speech — all working at first boot, no setup needed.

⚡

Dev tools pre-loaded

Git, Neovim, Docker, Ghostty terminal, Zsh with plugins, and more — configured and ready before you open a terminal.

🔄

Always up to date

Rolling release means you get updates as soon as they drop. One command keeps everything fresh.
