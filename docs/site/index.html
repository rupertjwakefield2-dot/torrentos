Getting started
Up and running in minutes.

Flash a USB, boot it up, and you're done. No partitioning wizards, no package selection screens.

1

Download the ISO

Grab the latest release from the releases page. Verify the SHA256 checksum if you want to be sure the file is intact.

2

Flash it to a USB drive

Use Rufus on Windows or Balena Etcher on any platform. Select your USB, pick the ISO, and write it. Use DD mode if Rufus asks.

3

Boot from USB

Plug in the USB and restart your PC. Press F12, F2, or Delete to open the boot menu (depends on your motherboard). Select the USB drive.

4

Install to disk

Run the installer from the desktop. Follow the prompts to set your username, disk, and timezone. Takes about 10 minutes on most hardware.
