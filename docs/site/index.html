Getting started

Up and running in minutes.

Flash a USB, boot it up, and you're done.

1

Download the ISO

Grab the latest build from the releases page. The ISO is around 3.9 GB — verify it with the SHA256 checksum listed alongside the download.

2

Flash to USB

Use Rufus on Windows or Balena Etcher on any platform. When Rufus asks for write mode, pick DD Image.

3

Boot from USB

Plug in the USB and restart. Press F12, F2, or Delete at startup to open your boot menu. Select the USB.

4

Install to disk

Open the installer from the desktop. Follow the prompts for disk, username, and timezone. Takes about 10 minutes. Then reboot.
