# TorrentOS

An Arch-based Linux distribution that combines macOS aesthetics, class-leading
accessibility, and Linux developer power — built on Hyprland.

[![Release](https://img.shields.io/github/v/release/rupertjwakefield2-dot/torrentos?label=latest)](https://github.com/rupertjwakefield2-dot/torrentos/releases/latest)
[![License](https://img.shields.io/badge/license-GPL--3.0-blue)](LICENSE)

---

## Features

| Area | What's included |
|------|-----------------|
| **Desktop** | Hyprland compositor · macOS-feel glass dock (nwg-dock) · Waybar status bar · swaync notification centre |
| **Launcher** | Rofi with fuzzy search, scrollbar, icon-aware app list and power menu |
| **Terminal** | Ghostty with JetBrains Mono Nerd Font, TorrentOS colour scheme, starship prompt |
| **Shell** | Zsh with eza, bat, fzf, zoxide, ripgrep, rich aliases and login banner |
| **Settings** | GTK4/libadwaita Settings app — Appearance, Display, Keyboard & Mouse, Network, Accessibility, Dev Mode, About |
| **Screenshot** | `torrentos-screenshot` GUI (region/full/window, delay, clipboard copy, Loupe preview) |
| **Update** | `torrentos-update-gui` — VTE-powered live update terminal with progress bar |
| **First boot** | GTK4 setup wizard (keyboard layout, timezone, locale, wallpaper) |
| **Install** | `torrentos-install-gui` — disk picker, internet check, install-to-disk UI |
| **Theming** | Dark palette (navy #0A1628 / blue #1E6FFF), Papirus-Dark icons, Bibata cursor, Inter & JetBrains fonts, Qt6/GTK unified |
| **Accessibility** | Orca screen reader, at-spi2, high-contrast, colour filters, espeak |
| **Dev Mode** | One-click toolchain: Rust · Go · JDK · Bun · Deno · VS Code · lazygit · k9s · Docker |

## Repo layout

```text
torrentos/
├── archiso/                     archiso profile consumed by mkarchiso
│   ├── profiledef.sh            ISO metadata and build flags
│   ├── packages.x86_64          packages installed into the live system
│   ├── pacman.conf              pacman config used inside the chroot
│   └── airootfs/                files copied verbatim into the live root
│       ├── etc/skel/            default user dotfiles (hypr, waybar, rofi, zsh…)
│       ├── usr/local/bin/       TorrentOS scripts (screenshot, update-gui, help, install)
│       └── usr/share/torrentos/ branding and assets
├── packages/                    custom PKGBUILDs hosted in the torrentos repo
│   ├── torrentos-base/          meta-package, branding, settings daemon
│   ├── torrentos-theme/         GTK/Qt/icon/cursor theme bundle
│   ├── torrentos-hyprland-config/ default Hyprland, Waybar, rofi, swaync, ghostty, zsh configs
│   ├── torrentos-first-boot/    GTK4 welcome wizard
│   ├── torrentos-settings/      GTK4 settings application
│   └── torrentos-tools/         GUI tools — screenshot, update, help
├── scripts/                     build helpers
│   ├── build-pkgs.sh            builds custom PKGBUILDs into ./repo/
│   ├── build-aur.sh             builds required AUR packages into ./repo/
│   ├── build-iso.sh             wraps mkarchiso
│   └── test-iso.sh              boots the newest ISO in QEMU
├── docs/
│   └── site/                    torrentos.org static site
└── repo/                        local pacman repo generated by build scripts
```

## Quick start (build from source)

You need an **Arch Linux host** (WSL2 with Arch, a VM, or bare metal). `mkarchiso`
requires a Linux kernel and loop devices — Windows-native builds are not supported.

```bash
sudo pacman -S --needed archiso base-devel git
git clone https://github.com/rupertjwakefield2-dot/torrentos
cd torrentos
./scripts/build-pkgs.sh   # builds custom packages into ./repo/
./scripts/build-aur.sh    # builds AUR packages into ./repo/
./scripts/build-iso.sh    # builds the final ISO via mkarchiso
```

The ISO is written to `./out/torrentos-<date>-x86_64.iso`.

See [docs/BUILDING.md](docs/BUILDING.md) for the full build guide.
